flexbox 指定一覧

1. display: flex;

親要素に「display: flex;」を指定すると、子要素の数に応じて親要素の幅を均等に分割かつ横伸縮する形で並ぶ。

1
2
3
4
5
6
                    
        // html
        <div class="flex-container">
            <div class="flex-items">1</div>
            <div class="flex-items">2</div>
            <div class="flex-items">3</div>
            <div class="flex-items">4</div>
            <div class="flex-items">5</div>
            <div class="flex-items">6</div>
        </div>
                    
                    
        /* css */
        .flex-container {
            display: flex;
        }

2. flex-direction（横並びと縦並び） 指定方法

親要素 flex-container に「flex-direction: column」を指定する。するとflexアイテムは上から下へ垂直方向に配置される。

2. flex-direction（横並びと縦並び） 指定方法

親要素 flex-container に「flex-direction: row-reverse」を指定する。するとflexアイテムはrowと逆、右から左へ水平方向に配置される。
