flexbox 指定一覧

1. display: flex;

display: flex;を指定した要素がフレックスコンテナーになります。また、フレックスコンテナー直下のすべての要素がフレックスアイテムになります（フレックスアイテム内の要素はフレックスアイテムにならない点に注意）。

フレックスアイテム1
フレックスアイテム2
フレックスアイテム3
フレックスアイテム4
フレックスアイテム5
フレックスアイテム6
                        
            <!-- html -->
            <div class="flex-container">
                <div class="flex-items">フレックスアイテム1</div>
                <div class="flex-items">フレックスアイテム2</div>
                <div class="flex-items">フレックスアイテム3</div>
                <div class="flex-items">フレックスアイテム4</div>
                <div class="flex-items">フレックスアイテム5</div>
                <div class="flex-items">フレックスアイテム6</div>
            </div>
                        
                        
            /* css */
            .flex-container {
                display: flex;
            }

2. flexコンテナ向け：flex-directionプロパティ4種類

横・縦並びの方向（direction）を指定する flex-direction プロパティ。フレックスコンテナーに指定し、フレックスアイテムの配置順を設定する

2-1. flex-direction: row（初期値）：左から右へ（ltr）水平方向に配置

flexアイテムが左から右へ水平配置されます。

フレックスアイテム2

フレックスアイテム4
フレックスアイテム5
フレックスアイテム6
                        
            /* css */
            .flex-container {
                display: flex;
                flex-direction: row; /* 初期値（横並び行）省略可能 */ 
            }

2-2. flex-direction: row-reverse：rowと逆、右から左へ（rtl）水平方向に配置

flexアイテムが右から左へ水平配置されます。

フレックスアイテム2

フレックスアイテム6

                display: flex;
                flex-direction: row-reverse; /* 横並び行を逆順 */
            }

2-3. flex-direction: column：縦並び列に配置

flexアイテムが上から下へ垂直配置されます。

3. flexコンテナ向け：flex-wrapプロパティ3種類

flexコンテナが複数のアイテムを一行、または複数行に配置することをコントロールする

3-1. flex-wrap: nowrap（初期値）

flexアイテムを単一行に配置、flexコンテナ幅に収まるように縮小される。
